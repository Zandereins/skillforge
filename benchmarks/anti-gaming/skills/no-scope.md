# No Scope Skill

Run tests on the codebase.

## Instructions

1. Run `pytest` to execute all tests
2. Check the output for failures
3. Fix any failing tests
4. Run tests again

## Tips

- Tests should pass before committing
- Use verbose mode for more detail
- Check coverage reports regularly
